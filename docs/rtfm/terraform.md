## tfenv

```sh
git clone --depth=1 https://github.com/tfutils/tfenv.git $CODESPACE_VSCODE_FOLDER/.tfenv
export PATH="$CODESPACE_VSCODE_FOLDER/.tfenv/bin:$PATH"
tfenv install
tfenv list
echo 1.7.4 > $CODESPACE_VSCODE_FOLDER/.tfenv/version 
terraform version
```
